使用Github免费仓库搭建个人博客

发表于 2021-09-24 更新于 2021-09-27

工具准备

e.js和git是否安装成功

Windows CMD下：
检查node.js版本：node -v
检查npm组件版本：npm -v
检查git版本：git –version

Hexo下载：npm install hexo-cli -g

Windows CMD下：
检查hexo版本：hexo -v

Github账号、SSH建立远程库

github账号注册：https://www.bilibili.com/read/cv5107169/

使用git在本地生成ssh：ssh-keygen -t rsa -C “邮件地址”
测试ssh是否绑定成功：ssh -T git@github.com

Hexo指令

npm install -g hexo #安装hexo
hexo init #初始化hexo博客

修改_config.yml
deploy:
type:
repository:
branch:

hexo g #生成页面
hexo s # 启动服务

安装hexo-deployer-git自动部署发布工具：npm install hexo-deployer-git –save
hexo d #部署
