使用Github免费仓库搭建个人博客

发表于 2021-09-24

一、工具准备：
Node.js下载：网址请参考安装教程
【它让JavaScript成为与PHP、Python、Perl、Ruby等服务端语言平起平坐的脚本语言。】
安装教程：https://www.cnblogs.com/liuqiyun/p/8133904.html

Git下载：https://github.com/git-for-windows/git/releases/download/v2.11.0.windows.3/Git-2.11.0.3-64-bit.exe
【Git是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理】
安装教程：https://www.cnblogs.com/xueweisuoyong/p/11914045.html

检查node.js和git是否安装成功，Windows CMD下：

检查node.js版本：node -v

检查npm组件版本：npm -v

检查git版本：git –version

Hexo下载：npm install hexo-cli -g

检查hexo是否安装成功，Windows CMD下：

检查hexo版本：hexo -v

二、注册Github账号
github账号注册：https://www.bilibili.com/read/cv5107169/

使用git在本地生成ssh：ssh-keygen -t rsa -C “邮件地址”
测试ssh是否绑定成功：ssh -T git@github.com

npm install -g hexo #安装hexo
hexo init #初始化hexo博客

修改_config.yml
deploy:
type:
repository:
branch:

hexo g #生成页面
hexo s # 启动服务

安装hexo-deployer-git自动部署发布工具：npm install hexo-deployer-git –save
hexo d #部署

hexo使用

Quick Start
创建一片博客
$ hexo new “My New Post”
More info: Writing

启动服务
$ hexo server
More info: Server

生成页面文件
$ hexo generate
More info: Generating

部署到远程站点
$ hexo deploy
More info: Deployment
